# Sanskrit And Bengali Transliteration

For SEO purpose it's preferable not to have diacritics on the web. However having IAST transliteration for academic purpose is nice too. So the combined aproach is used. For sanskrit / Bengali use IAST version, combined with simplified version without diacritics. Markup as follows (IASt version should come first):

```
{{ahaṁ tvāṁ sarva-pāpebhyo|aham tvam sarva-papebhyo}}
{{mokṣayiṣyāmi mā śucaḥ|mokshayishyami ma shucah}}
```
